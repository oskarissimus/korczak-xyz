Zaawansowane narzędzia web-testera

Na kurs składa się 12 1-godzinnych indywidualnych spotkań z kursantem. Terminy spotkań są ustalane indywidualnie. Celem kursu jest zapoznanie z zaawansowanymi narzędziami stosowanymi w pracy web-testera. Oprócz materiału omawianego podczas spotkań prowadzący przedstawia również dodatkowe materiały do pracy własnej, które mają na celu utrwalić wiedzę i umiejętności kursanta.

Tematyka kursu:

  1. Narzędzia deweloperskie w Chrome
  2. Postman
  3. Wprowadzenie do Python + Selenium

Tematy spotkań

  1. Omówienie możliwości narzędzi deweloperskich w Chrome
  2. Badanie elementu i dynamiczne modyfikowanie jego właściwości
  3. JavaScript console
  4. Zakładka network - omówienie budowy pakiety HTTP
  5. Zakładka network - analiza i praktyczne wykorzystanie
  6. Postman - wprowadzenie i omówienie możliwości
  7. Postman - tworzenie requestów
  8. Postman - workspaces, collections
  9. Python - wprowadzenie
  10. Python - zmienne, instrukcje warunkowe
  11. Python - pętle, funkcje
  12. Python - konfiguracja środowiska selenium
